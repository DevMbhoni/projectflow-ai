# ProjectFlow AI

## AI-Powered Project Management System

ProjectFlow AI is an AI-powered project management system developed as part of the **CAPACITI AI Skill Accelerator Programme**.

This project was created to apply the key concepts learned during the programme, including AI fundamentals, productivity with AI tools, prompt engineering, responsible AI use, and applying modern AI solutions to real-world workplace problems.

The project was built using **AI-assisted development**, mainly through **Lovable AI**, while applying concepts from the **Google AI Essentials** learning modules.

---

## Project Overview

In modern workplaces, professionals and teams often spend a lot of time on repetitive administrative tasks such as writing emails, summarizing meeting notes, planning schedules, preparing project updates, and creating reports.

ProjectFlow AI addresses this challenge by combining a normal project management system with integrated AI-powered productivity features.

The system helps users manage projects and tasks while using AI to generate structured workplace outputs such as meeting summaries, task plans, project reports, and professional emails.

---

## Problem Statement

Professionals across different industries spend significant time on repetitive workplace tasks. These tasks are important, but they often reduce productivity and take time away from more valuable work.

Common workplace challenges include:

- Writing professional emails manually
- Summarizing long meeting notes
- Identifying decisions and action items
- Planning daily or weekly tasks
- Preparing project progress reports
- Tracking risks, blockers, and deadlines
- Communicating updates clearly with clients, managers, and teams

Manual handling of these tasks can lead to delays, poor communication, missed responsibilities, and reduced productivity.

---

## Solution Overview

ProjectFlow AI is a web-based project management prototype that integrates AI features into everyday project workflows.

Instead of creating a general chatbot-only system, this project demonstrates how AI can be embedded into a practical workplace system.

The system allows users to:

- Create and manage projects
- Create and manage tasks
- Summarize meeting notes using AI
- Generate task plans using AI
- Draft professional workplace emails
- Generate project status reports
- Review AI outputs before using them

The AI features are designed to assist users and improve productivity, while still requiring human review and validation.

### 3. Discover the Art of Prompting

Prompt engineering played an important role in this project.

The prompts were designed, tested, and refined to produce clear, useful, and structured outputs.

The prompts were improved to:

- Use the user’s actual input
- Avoid generic responses
- Extract names, dates, deadlines, tasks, risks, and decisions
- Avoid inventing missing information
- Return organized workplace outputs
- Match the selected tone and audience
- Improve accuracy and usefulness

## Sample Prompts

### Meeting Notes AI Prompt

```text
You are an AI meeting notes assistant.

Summarize the meeting notes below into a clear workplace summary.

Meeting notes:
[raw meeting notes]

Return the result in this structure:
1. Meeting summary
2. Key discussion points
3. Decisions made
4. Action items
5. Responsible person for each action
6. Deadlines
7. Follow-up recommendations

Do not invent missing information. If a deadline or responsible person is unclear, write “Not specified”.
```

### AI Task Planner Prompt

```text
You are an AI productivity planner.

Create a structured work plan based on the tasks below.

Tasks:
[user tasks]

Available working hours:
[hours]

Deadlines:
[deadlines]

Prioritize the tasks using urgency, importance, deadlines, and available working hours.

Return:
1. Priority ranking
2. Suggested schedule
3. Reason for each priority
4. Time optimization tips
5. Risks or blockers
6. Recommended next steps

If total task effort exceeds available time, clearly show what should be done first and what should be moved later.
```

### Smart Email Generator Prompt

```text
You are a professional workplace communication assistant.

Generate a professional email using the following details:

Purpose:
[email purpose]

Audience:
[client / manager / team / HR / stakeholder]

Tone:
[formal / friendly / persuasive / apologetic / concise]

Recipient name:
[recipient name]

Key details:
[user details]

The email must include:
- Subject line
- Greeting
- Professional body
- Clear call to action
- Polite closing

Do not copy the purpose directly. Transform it into natural professional language.
Avoid exaggeration, unclear claims, or unverified information.
```

### Project Report Prompt

```text
You are an AI project reporting assistant.

Create a structured project status report from the progress notes below.

Progress notes:
[user progress notes]

Return:
1. Executive summary
2. Completed work
3. Work in progress
4. Pending work
5. Risks or blockers
6. Recommendations
7. Next steps

Do not invent project facts. If information is missing, write “Not specified”.
```
